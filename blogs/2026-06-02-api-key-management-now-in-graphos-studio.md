API keys are fundamental to securing GraphQL infrastructure, but their management has traditionally been scattered across different tools and locations. Previously, subgraph keys, operator keys, and SCIM keys were only accessible through Rover CLI and the Platform API, with limited options for handling compromised or expired credentials. This new unified interface in GraphOS Studio consolidates all key types in one place, offering improved visibility and control.
